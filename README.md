# CodeIgniter3-CRUD
Projeto CRUD simples de gerenciamento de empregados usando:
 - CodeIgniter 3,
 - PHP (8.1), 
 - MySQL,
 - Javascript,
 - JQuery, 
 - Ajax,
 - Bootstrap 5.

![image](https://github.com/kariluss/CodeIgniter3-CRUD/assets/62659613/e8a4de57-b62b-497e-917d-83cc9b4bbc61)

Criado e testado num servidor Apache.
